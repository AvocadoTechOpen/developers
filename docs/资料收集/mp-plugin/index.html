小程序插件

资料收集小程序插件是鳄梨科技提供的微信小程序端资料收集服务，致力于用AI解决大规模人力资源问题，用于提供给第三方小程序调用。

插件使用说明​

添加插件​

在使用插件前，首先要在小程序管理后台的“设置-第三方设置-插件管理”中添加插件。开发者可登录小程序管理后台， 通过 appid [wx6a8e6a71530652d5] 查找插件并添加

引入插件​

主包中引入插件​

{
  "plugins": {
    "formPlugin": {
      "version": "1.0.1",
      "provider": "wx6a8e6a71530652d5",
      "export": "index.js"
    }
  }
}

分包中引入插件​

主包尺寸超过2M时，使用

{
  "subpackages": [
    {
      "root": "formPlugin",
      "name": "formPlugin",
      "pages": [
        "leadPage/leadPage"
      ],
      "plugins": {
        "formPlugin": {
          "version": "1.0.2",
          "provider": "wx6a8e6a71530652d5",
          "export": "index.js"
        }
      }
    },
  ]
}

小程序进入插件​

<!-- 路由组件方式 -->
<navigator url="plugin://formPlugin/index> navigator: Navigate to Plugin page </navigator>

<!-- methods方式 -->
<button bindtap="enterPlugin">进入资料收集</button>
const plugin = requirePlugin('formPlugin')

Page({
  data: {},
  onLoad() {
    wx.redirectTo({
      url: `plugin://formPlugin/index`
    })
  }
})

小程序与插件交互​

小程序给插件传递数据​

  • 使用配置文件
// videoInterview/index.js
module.exports = {
  initData: {
    customNavigateKey: 'eli', // 自定义跳转判定条件
    nextProcessToken: 'wjhYdPltXOkPlIRNYCGiRGUW2D', // 面试系统内获取
    curSessionToken: '8bwEU74R19AzfiUQW6ZVRGFMH7', //岗位Token
    authToken: 'bhfABiH0knGrtCGItguNjZoEiXTRvcy1RfmY', //用户的token
  }
}
  • 使用插件提供的API动态设置
// plugin api
module.exports = {
  /**
   * @description: 给插件传递的配置类数据
   * @param {String} nextProcessToken 面试token
   * @param {String} curSessionToken 岗位token
   * @param {String} authToken 用户登录token
   */
  setConfig(data = { nextProcessToken: '', curSessionToken: '', authToken: '' }) {},
}

// 使用示例
const plugin = requirePlugin('formPlugin')

// 会覆盖配置文件
plugin.setConfig({
  nextProcessToken: 'xxx',
  curSessionToken: 'xxx',
  authToken: 'xxx'
})

插件与小程序通信方式​

// 插件与小程序通信，通过eventBus进行，小程序接收插件的消息，需要进行事件注册

const plugin = requirePlugin('formPlugin')

registerEvents() {
  // 小程序给插件传递消息
  plugin.eventBus.emit('mini', {})
  // 小程序接收来自插件的消息
  plugin.eventBus.on('plugin', (data) => {
    console.log('插件来消息了>>>>>>', data)
  })
},
