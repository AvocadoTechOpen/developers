前端实时面试 UI sdk

最后更新时间: 2021年12月24日

概述:​

本文档针对邀约面试结果呈现进行了标准封装,封装了一些基础交互.如视频播放和面试文字展示.帮助用户快速集成对应的实时面试管理功能.

UI 标准展示:​

  • 推荐单独页面展示或者嵌入一个比较大的容器来显示.

  • 整体显示宽度最大586px 最小宽度360px,在428px的时候会切换到移动端样式.

接入方式​

通用接入方式​
  • 通过 cdn 引入对应的 js文件
<script src="https://webcdn.fbmms.cn/sdk/widget.js"></script>
  • 配置获得授权方法

根据接口提供的 accesstoken 存在2个小时过期的机制,此处需要业务方自行完成 token 获取和保持有效性

<script>
AvocadoInterviewUI.getAccessToken = ()=>{
   return "accesstoken"
}
</script>
  • 执行render 操作渲染对应的容器中.可以多次调用
<script>
//interviewerToken 邀约 token
//selector 要渲染的 Container的 id
    AvocadoInterviewUI.render(selector, {token: "interviewerToken"})
//示例
// AvocadoInterviewUI.render(document.getElementById("container"),{ token: "3tgqt9LByQMmOHcY00GJR3VZZK" })
</script>
