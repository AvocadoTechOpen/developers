web端插件

实时视频面试web端插件是鳄梨科技提供的【实时视频面试+实时视频面试报告】的服务

✨ 特性​

  • 简单易用
  • 包含的组件有实时视频面试、实时视频面试报告
  • 使用 TypeScript 构建，提供完整的类型定义文件

📦 安装​

// npm
npm i eli-live-interview --save

// yarn
yarn add eli-live-interview

🔨 配置​

import { avocadoLiveInterview } from 'eli-live-interview';

const App: React.FC = () => {
  useEffect(() => {
    avocadoLiveInterview.init({
      server_url: '', // 接口域名地址
      auth_callback: () => new Promise((resolve, reject) => { // 授权认证回调 return Promise
        resolve({
          accessToken: '', // 必须返回accessToken为接口授权认证
          expireTime: 0, // accessToken过期时间（可以不返回）
        });
      }),
    });
  }, []);
}

实时视频面试​

实时视频面试组件：功能包括面试等待、面试环境检测、面试间、面试间状态（离开、结束）

Preview​

Example​

import { LiveRoom, useLiveRoom } from 'eli-live-interview';

const { onClickUser, onRoomStatusChange, ...LiveRoomProps } = useLiveRoom({ interviewToken, userToken });

// 当前点击的用户，供业务方做交互处理
const onClickUser = async (userInfo) => {
  console.log(userInfo);
};

// 面试间状态变化
const onRoomStatusChange = async (roomStatusInfo) => {
  console.log(roomStatusInfo);
};

return (
  <LiveRoom {...LiveRoomProps} />
)

API​

参数说明类型必传默认值
interviewToken面试间tokenstring是-
authToken鉴权tokenstring是-
roomWaitingContentConfig面试等待页内容配置RoomWaitingContentConfig否[]
virtualBackgroundConfig企业内置的虚拟背景配置VirtualBackgroundConfig否{ virtualOpen:0, canChangeVirtualBackground: 1, virtualBackgroundList: [] }

LiveRoomInstance​

名称说明类型
onClickUser获取当前点击的用户信息() => UserInfo
onRoomStatusChange面试间状态变化() => RoomStatusInfo

RoomWaitingContentConfig​

名称说明类型
category分类名称，用于tab显示string
contentType内容类型string
roomWaitingContent内容Array

RoomWaitingContent​

名称说明类型
label标题string
optionType选项类型string
value选项值string

VirtualBackgroundConfig​

名称说明类型
virtualOpen背景设置类型
canChangeVirtualBackground是否允许面试官更改默认背景图number
virtualBackgroundList角色类型number

VirtualBackgroundList​

名称说明类型
backgroundUrl企业面试官默认背景图片string
isUse是否当前使用number

UserInfo​

名称说明类型
userToken平台面试用户tokenstring
name面试用户姓名string
mobile面试用户手机号string
email面试用户邮箱string
platformUserId第三方用户标识string
userRole角色类型number

RoomStatusInfo​

名称说明类型
roomStatus面试间状态类型string

实时视频面试报告​

实时视频面试报告组件用于提供面试视频回放、面试结果分析、面试问题提问

Preview​

Example​

import React, { FC, useState, useCallback } from 'react';
import { Box } from 'fbm-ui';
import { LiveInterviewReport, useLiveInterviewReport } from 'eli-live-interview';

const Report: FC = () => {
  const interviewToken = ''; // 面试token
  const canEditInterviewSchema = 1; // 是否可以编辑面试总览
  const showInterviewSchema = 2; // 是否显示面试总览

  const liveInterviewReport = useLiveInterviewReport({
    interviewToken,
    canEditInterviewSchema,
    showInterviewSchema,
    onLoad: params => {
      console.log(params);
    },
  });

  return (
    <Box sx={{ width: '100%', height: '100%', overflow: 'hidden' }}>
      <Box sx={{ width: '584px', height: '100%', margin: '0 auto' }}>
        <LiveInterviewReport {...liveInterviewReport} />
      </Box>
    </Box>
  );
};

export default Report;

API​

参数说明类型必传默认值
interviewToken面试tokenstring是-
userToken用户tokenstring是-
canEditInterviewSchema是否可以编辑面试总览boolean否1
showInterviewSchema是否展示面试总览boolean否1
onLoad报告数据获取成功后回调事件(values: LiveInterviewReportLoadedValues) => void否-

全局类型常量​

角色类型​

1: 候选人
2: 面试官

内容类型​

LIST: 列表

选项类型​

TEXT: 文本
IMAGE: 图片
VIDEO: 视频

背景设置类型​

0: 关闭
1: 背景虚化
1: 背景图片

背景图是否当前使用​

0: 不是
1: 是

更改虚拟背景​

1: 默认可以更改
2: 不可以更改

面试间状态类型​

CANCEL: 沟通已取消
INVALID: 链接已失效
KICK: 被迫下线
LEAVE: 用户离开
END: 面试结束

是否可以编辑面试总览​

1: 默认可以更改
2: 不可以更改

是否展示面试总览​

1: 显示
2: 不显示

HTTP状态码​

API采用标准HTTP错误码来表示接口的错误:

状态码含义
200OK -- 请求成功
400Bad Request -- 接口请求参数错误.
401Unauthorized -- 认证信息非法.
403Forbidden -- 所访问的资源拒绝访问.
404Not Found -- 访问的资源不存在.
405Method Not Allowed -- 错误的请求Method.
500Internal Server Error -- 服务器内部错误.
503Service Unavailable -- 服务器暂时不可用.

业务状态码​

状态码含义
0OK -- 请求成功
其他codeERROR -- 请求失败
