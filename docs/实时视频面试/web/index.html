web端插件

实时视频面试web端插件是鳄梨科技提供的【实时视频面试+实时视频面试报告】的服务

✨ 特性​

  • 简单易用
  • 包含的组件有实时视频面试、实时视频面试报告
  • 使用 TypeScript 构建，提供完整的类型定义文件

实时视频面试​

实时视频面试组件：功能包括面试等待、面试环境检测、面试间、面试间状态（离开、结束）

Preview​

API​

参数说明类型必传默认值
interviewToken面试间tokenstring是-
authToken鉴权tokenstring是-
positionInfo面试等待页内容配置PostionInfo否[]
backgroundInfo企业内置的虚拟背景配置BackgroundInfo否
onClickUser获取当前点击的用户信息(token:string)=>void否

PostionInfo​

名称说明类型
positionName岗位名称string
positionDesc岗位简介string
positionToken岗位tokenstring
treatments岗位福利string[]
requirement岗位要求string
cover岗位封面Cover
resumeImg简历图片string

Cover​

名称说明类型
videoUrl视频地址string
defaultVideoUrl默认视频地址string
shareCoverImg分享封面图string

BackgroundInfo​

名称说明类型
virtualOpen背景设置类型number
enterpriseImage企业面试官默认背景图片string
backgroundImages虚拟背景图片列表BackgroundImage[]
userType角色类型number
canChangeVirtualBackground是否允许面试官更改默认背景图number

BackgroundImage​

名称说明类型
imageUrl虚拟背景图片地址string
isUse背景图是否当前使用number
token背景图tokenstring

实时视频面试报告​

实时视频面试报告组件用于提供面试视频回放、面试结果分析、面试问题提问

Preview​

📦 安装​

// npm
npm i eli-live-interview-report --save

// yarn
yarn add eli-live-interview-report

Example​

import React, { FC, useState, useCallback } from 'react';
import { Box } from 'fbm-ui';
import { LiveInterviewReport, useLiveInterviewReport } from "eli-video-interview-report";

const Report: FC = () => {
  const interviewToken = ''; // 面试token
  const userToken = ''; // 用户token
  const host = 'https://fbm-api-test-gw.fbmms.cn/video-interview/api'; // api地址
  const authToken = '' // 企业的鉴权Token
  const debug = true; // 是否开启接口 debug

  const liveInterviewReport = useLiveInterviewReport({
    host,
    interviewToken,
    authToken,
    userToken,
    debug
  });

  return (
    <Box sx={{ width: '100%', height: '100%', overflow: 'hidden' }}>
      <Box sx={{ width: '584px', height: '100%', margin: '0 auto' }}>
        <LiveInterviewReport {...liveInterviewReport} />
      </Box>
    </Box>
  );
};

export default Report;

API​

参数说明类型必传默认值
authToken企业鉴权 tokenstring是
interviewToken面试tokenstring是-
userToken用户tokenstring是-
host服务器地址string否https://fbm-api-test-gw.fbmms.cn/video-interview/api
debug服务端数据校验提示Boolean否false

全局类型常量​

角色类型​

1: 候选人
2: 面试官

内容类型​

LIST: 列表

选项类型​

TEXT: 文本
IMAGE: 图片
VIDEO: 视频

背景设置类型​

0: 关闭
1: 背景虚化
1: 背景图片

背景图是否当前使用​

0: 不是
1: 是

更改虚拟背景​

1: 默认可以更改
2: 不可以更改

面试间状态类型​

CANCEL: 沟通已取消
INVALID: 链接已失效
KICK: 被迫下线
LEAVE: 用户离开
END: 面试结束

是否可以编辑面试总览​

1: 默认可以更改
2: 不可以更改

是否展示面试总览​

1: 显示
2: 不显示

HTTP状态码​

API采用标准HTTP错误码来表示接口的错误:

状态码含义
200OK -- 请求成功
400Bad Request -- 接口请求参数错误.
401Unauthorized -- 认证信息非法.
403Forbidden -- 所访问的资源拒绝访问.
404Not Found -- 访问的资源不存在.
405Method Not Allowed -- 错误的请求Method.
500Internal Server Error -- 服务器内部错误.
503Service Unavailable -- 服务器暂时不可用.

业务状态码​

状态码含义
0OK -- 请求成功
其他codeERROR -- 请求失败
